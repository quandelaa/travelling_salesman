# Travelling Salesman

A TSP solver using a tile-based system

## Description

This project originally started because I wanted to prove to myself if I was really capable of doing something without help of an already structured project (that I only needed to fill in functions). I wanted to make my own codebase and experience how it really feels like.

---

Essentially solves a given TSP problem (start city, *cities, end city) from the user by a grid tile using simulated annealing and Manhattan Distance as its heuristic.

## Getting Started

### Usage

```
python runner.py
```

### How It Works

1. Put a starting point in the grid,
2. Put some cities,
3. Put an end point

(order not forced by the way)

4. Click the "SOLVE" button in the "Grid Manager" window
5. Wait
6. Solved (result may not be the best path)
